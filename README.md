### 🌟 Overview

**CareerShift AI** is a cutting-edge platform designed to help professionals navigate the ongoing AI revolution. If your career is at risk of automation, CareerShift AI calculates your exact displacement risk using **GPT-4o**, recommends AI-proof career alternatives, and generates a personalized week-by-week reskilling roadmap to get you there.

### 🚀 Local Setup & Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/careershift-ai.git
   cd careershift-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file in the root directory and add:
   ```env
   # Database
   DATABASE_URL="your-postgresql-url"
   
   # Authentication Secrets
   JWT_SECRET="your-generated-secret"
   JWT_REFRESH_SECRET="your-generated-secret"
   NEXTAUTH_SECRET="your-generated-secret"
   
   # AI
   GEMINI_API_KEY="your-gemini-key"
   ```

4. **Run Database Migrations**
   ```bash
   npx prisma db push
   ```

5. **Start the Development Server**
   ```bash
   npm run dev
   ```
   *The app will be running at [http://localhost:3000](http://localhost:3000)*

### 🌐 Deployment Guide (Vercel)

This project is fully optimized for [Vercel](https://vercel.com/). 

1. Push your code to a GitHub repository.
2. Import the project into your Vercel Dashboard.
3. In the Vercel deployment settings, paste all the environment variables from your `.env.local`.
4. Click **Deploy**. Vercel will automatically build the Next.js app and provide you with a live URL.
